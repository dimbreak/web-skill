# Demos

This directory contains fully independent Vite demo apps.

## Included demos

- `battleship-with-web-skill`
- `2048-with-web-skill`

Each demo has:

- its own `package.json` and lockfile
- its own local development/build scripts
- its own deployment workflow to Vercel
- independent path-based CI/CD triggers (deploy only when that demo changes)
