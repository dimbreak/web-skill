# Code of Conduct

We want `web-skill` to be a respectful and practical open-source project for builders working on real workflows.

## Our expectations

Please:

- be respectful and constructive
- assume good intent, but be clear about risks and tradeoffs
- focus feedback on ideas, code, documentation, and behavior, not on people
- welcome questions from contributors with different levels of experience
- keep discussions grounded in the project goals and user needs

## Unacceptable behavior

This project does not welcome:

- harassment, intimidation, or personal attacks
- discriminatory or hateful language
- bad-faith trolling, baiting, or repeated hostility
- publishing private information without permission
- spam or intentionally disruptive behavior

## Maintainer responsibility

Maintainers may edit, remove, or reject comments, issues, pull requests, and other contributions that do not align with this code of conduct.

## Reporting

If you experience or observe a serious problem, contact [dillion@dimbreak.com](mailto:dillion@dimbreak.com).
